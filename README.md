# InceptionVision
Demo app using CoreML, Vision and Inceptionv3 model 

## Requirements
- Xcode 9 or later
- Swift 4.0 or later
- iOS 11 or later (* a device is required for this demo)

## Installation
1. `git clone https://github.com/Lybron/inception-vision.git`
2. Open `InceptionVision.xcodeproj`
3. Run on device.
